My Projects

I have worked on a variety of different projects during my studies and independently. Check out some of the cool work I have done below and be sure to check back as I add more interesting projects!

Full Stack Blogging Application

Created a blogging web app using Python and Flask. The app allowed users to create their own account and create, update and delete blog posts. SQLAlchemy was used to create a SQLite database to store data for the blog posts as well as user data. The app featured a responsive front end design created using the Bootstrap framework with HTML and CSS.

Propulsion and Control System Design for Mobile Aquaculture Cage

My University capstone project. Tasked with addressing issues of health concerns regarding stationary aquaculture farms, we designed a concept for a mobile aquaculture cage. Developed a model of the dynamics of an immersed aquaculture cage which was implemented into MATLAB and Simulink to simulate dynamic behaviour. A PID control system was then implemented into the model and tuned to evaluate the feasibility and effectiveness of a generalized control system for a mobile aquaculture cag.

Neutralizing a Light Source with a Programmable Robot

Using Arduino, programmed a LynxBot mobile robot with a mounted Arduino microcontroller board to follow a wall, scan room to locate a randomly placed light source, and then approach and block the light source. This was done by integrating a Sharp infrared sensor to develop a wall following algorithm using a PID control system. A photoresistor and servo motor was integrated to scan the room and locate light source.

Reverse Engineering a Blackbox System with Control System Design

The purpose of this project was to develop a controller for a black box system. To do this, we developed a heuristic transfer function to model the input/output behaviour of the system. This was done using MATLAB to evalauate the system for LTI and white noise properties, and evaluating the systems frequency response with filtered output to construct a Bode plot, which allowed us to estimate the transfer function. This was then implemented into Simulink in a closed loop control system and a PID controller was designed to meet provided metrics in steady state error, rise time, overshoot, and settling time.
